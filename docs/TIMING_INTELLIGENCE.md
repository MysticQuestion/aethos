# Timing Intelligence

Aethos timing windows are event windows, not absolute predictions.

Implemented:

- Transit event generation from deterministic demo provider output
- Theme contribution mapping
- Theme aggregation
- Intensity score
- Confidence score
- Suggested reflection
- Recommended action experiment
- Raw source event table in `/timing-lab`

Themes include Agency, Inner Life, Structure, Expression, Relational Patterns, Material Stability, Conflict, Renewal, Visibility, Rest, and Decision Pressure.

Demo mode warning:

Current timing windows are deterministic samples. Production timing requires a verified ephemeris backend.
