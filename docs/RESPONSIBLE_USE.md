# Responsible Use

Aethos is a symbolic-intelligence, self-knowledge, journaling, reflection, and decision-support platform.

It does not provide:

- Medical advice
- Legal advice
- Financial advice
- Psychiatric diagnosis
- Guaranteed prediction
- Deterministic fate claims

Recommended language:

- Patterns
- Timing windows
- Interpretive lenses
- Confidence
- Reflection
- Observed correlation
- Personal synthesis

Avoid language such as "guaranteed prediction," "unlock your destiny," "AI psychic," or any claim that removes user agency.
